How To Run Project?
To run this project, you must have installed a virtual server i.e XAMPP on your PC (for Windows). This Pharmacy Management System in PHP with source code is free to download, Use for educational purposes only!

After Starting Apache and MySQL in XAMPP, follow the following steps

1st Step: Extract file
2nd Step: Copy the main project folder
3rd Step: Paste in xampp/htdocs/

Now Connecting Database

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on databases tab
6th Step: Create database naming “pms” and then click on import tab
7th Step: Click on browse file and select “pms.sql” file which is inside “database” folder
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/pharmacy_Management_System/
